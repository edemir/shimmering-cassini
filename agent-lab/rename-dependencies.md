# 📎 Rename Dependencies (Reference Images)

> This page previously contained embedded base64 image data from the original lab document export.
> Those images depicted the Cloud Console UI steps for renaming `requirements.txt` to `dependencies.txt`.

<div class="tip-box info">

**Note:** If you need to rename `requirements.txt`, simply run:

</div>

```bash {codejar-readonly}
mv requirements.txt dependencies.txt
```

Or use the Cloud Shell Editor's file explorer to right-click and rename the file.
