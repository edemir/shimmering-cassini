# 📊 07 - Create a Data Agent



In this step, you'll create a **Data Agent** in BigQuery's Agent Hub and connect it as a remote tool.

## Set up the Data Agent

1. Visit the [BigQuery Agent Hub](https://console.cloud.google.com/bigquery/agents_hub).
2. Enable the APIs if needed.
3. Go to **"Agent Catalog"** from the top of the screen.
4. Click on the ⋮ menu of **"The Look Ecommerce"** agent and select **Customize**.
5. Change agent name to "**My The Look Ecommerce**"
5. **Save and publish** the agent.
7. Click on **"Share"** button and share the agent with Agent Engine.  Get Agent Service Account ID from the Agent Engine console and add it as a <strong>`Gemini Data Analytics Data Agent User`
</strong>.

## Test the Data Agent

In BigQuery Agent chat, ask the agent what it can do to verify it's working.  Ask follow up questions to see if it can answer them correctly.
