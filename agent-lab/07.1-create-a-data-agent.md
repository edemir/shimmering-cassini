# 📊 07.1 - Create a Data Agent

<div class="duration">⏱ ~10 minutes</div>

In this step, you'll create a **Data Agent** in BigQuery's Agent Hub and connect it as a remote tool.

## Set up the Data Agent

1. Visit the [BigQuery Agent Hub](https://console.cloud.google.com/bigquery/agents_hub).
2. Enable the APIs if needed.
3. Go to **"Agent Catalog"** from the top of the screen.
4. Duplicate the **"The Look Ecommerce"** agent.
5. **Save and publish** the agent.

## Share with Agent Engine

<div class="tip-box info">
  <span class="icon">⚠️</span>
  <div>
    <strong>Important:</strong> Share the data agent with Vertex Agent Engine's service account — copy the <strong>"identity"</strong> from the Agent Engine console and add it as a <strong>viewer</strong>.
  </div>
</div>

## Test the Data Agent

In the chat, ask the agent what it can do to verify it's working.
